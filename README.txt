# Project Title

## Getting Started & Disclaimer

* Disclaimer: the funcionalities required work for digits whose first sum is at most 2 digits.Nevertheless the logical implementation of client-server socket follow the assignment requirements.

The project comes along with the make file, just run it and then execute the server to start listening. Afterwards execute the client.

### Prerequisites

You may as well want to install * make * in your machine if you haven't already
